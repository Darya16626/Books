Интернет-магазин книг на Django

Разработка web-сайта интернет-магазина книг с использованием Django и базы данных PostgreSQL. Проект реализует функционал для клиентов, администраторов и менеджеров с комплексным набором страниц и API для товаров, акций и заказов. Поддержка тёмной и светлой тем, система регистрации, авторизации и восстановления пароля.

Инструкция по запуску
Создать виртуальное окружение и активировать его:
python -m venv .venv
.venv\Scripts\activate

Перейти в папку проекта:
cd myproject

Запустить сервер разработки:
python manage.py runserver

История разработки

Неделя 1. Постановка задачи и выбор предметной области
Выбран проект интернет-магазина книг.
Определены цели: удобная покупка книг с акциями, личным кабинетом и чатом поддержки.
